# Mind Harbor / 心港

一个面向精神障碍康复支持场景的安全优先型辅助系统原型。

## 项目定位

Mind Harbor 不是 AI 医生，也不是自动诊断/自动改药系统。
它是一个 **康复支持系统**，帮助用户：

- 了解疾病与药物基础知识
- 记录每日状态
- 接收吃药提醒
- 记录副作用与服药依从性
- 获得低风险、温和、具体的小步行动建议
- 在高风险状态下转入危机支持与现实求助流程
- 为复诊准备结构化摘要

## 核心原则

1. 安全优先，宁可保守
2. 支持，不诊断
3. 不提供个体化改药建议
4. 危机处理高优先级
5. 先做低风险高价值功能
6. 通用底座 + 病种模式
7. 温和、不说教的语言
8. 强隐私与可删除性

## MVP 范围

- 用户档案与病种模式（抑郁/焦虑/双相基础版）
- 每日状态记录
- 药物与服药计划管理
- 服药提醒与确认
- 药品基础知识卡
- 风险分级与危机提示
- 支持性回应与微行动建议
- 复诊摘要导出

## 非目标

- 自动诊断
- 代替医生调整药物
- 生成处方
- 提供高风险心理治疗流程
- 在危机中仅靠 AI 独立处置

## 本地运行

```bash
cd project
npm start
```

打开：`http://localhost:3030`

- 删除药物 / 删除记录
