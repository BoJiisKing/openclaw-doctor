# 产品蓝图

## 1. 使命

帮助精神障碍或长期精神健康困难用户，在困难时期获得结构化支持，更稳定地生活、治疗与求助。

## 2. 目标用户

### 第一阶段核心用户
- 抑郁障碍/持续性低落用户
- 焦虑障碍/惊恐/强迫相关困扰用户
- 双相障碍基础管理用户

### 第二阶段扩展用户
- 家属/照护者
- 有长期复发史、需要复诊管理的用户

## 3. 用户核心问题

1. 不理解病情与治疗流程
2. 漏药、乱药、停药风险
3. 不会稳定记录状态
4. 情绪低谷/焦虑发作时不知道下一步做什么
5. 复诊时无法准确回忆情况
6. 家属不会支持，沟通冲突多

## 4. 产品价值

- 帮助理解：疾病与药物知识更清晰
- 帮助坚持：提醒和服药记录提升依从性
- 帮助觉察：状态变化可见化
- 帮助行动：把“熬过去”变成“先做一步”
- 帮助求助：危机时把用户引到现实世界支持

## 5. 主要功能

### 5.1 入门与建档
- 选择模式：抑郁 / 焦虑 / 双相
- 设置治疗阶段与关注重点
- 设置紧急联系人（可选）
- 设置服药时间

### 5.2 每日状态追踪
通用维度：
- 心情
- 焦虑
- 精力
- 睡眠
- 食欲
- 社交意愿
- 备注

扩展维度：
- 自伤/绝望想法
- 语速加快/兴奋/冲动
- 明显刺激回避或惊恐发作

### 5.3 服药管理
- 添加药物与用法说明（用户手动填写）
- 服药提醒
- 已服/漏服记录
- 副作用记录
- 近 7/14 天依从性

### 5.4 药物知识卡
- 药物类别
- 常见用途（通用说明）
- 常见副作用
- 注意事项
- 需要与医生沟通的情况

### 5.5 支持与建议
- 基于状态生成低风险支持性回应
- 输出 1~3 个微行动建议
- 不下诊断，不指导改药

### 5.6 危机支持
- 风险信号识别
- 显示求助建议
- 提示联系家属/医生/当地急救资源
- 建议不要独处 / 远离危险物品

### 5.7 复诊摘要
自动汇总：
- 过去 7/14 天情绪、睡眠、焦虑变化
- 漏药情况
- 记录到的副作用
- 用户想问医生的问题

## 6. 页面结构

1. 首页仪表盘
2. 今日打卡页
3. 服药页
4. 知识库页
5. 记录历史页
6. 危机支持页
7. 复诊摘要页
8. 设置页

## 7. MVP 成功标准

- 用户能完成每日记录
- 用户能收到并确认服药提醒
- 系统能做基础风险分级
- 系统能生成安全的支持性文本
- 系统能导出可读的复诊摘要

## 8. 后续路线图

### Phase 2
- 家属协作模式
- 病程趋势图
- 更细的触发因素记录
- 医生沟通问题生成器

### Phase 3
- 内容审核后台

- 更稳健的规则引擎
