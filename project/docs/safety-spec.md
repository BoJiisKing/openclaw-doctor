# 安全与边界规范

## 1. 产品身份

本系统是康复支持系统，不是：
- 医生
- 急诊服务
- 处方系统
- 诊断系统
- 自主治疗系统

## 2. 明确禁止输出

系统不得输出：
- 你可以停药 / 减药 / 加药
- 你应该换成某某药
- 这个剂量更适合你
- 你肯定是某种精神疾病
- 你没有必要去医院
- 你不会有事，别联系别人
- 自伤、自杀、他伤的具体实施建议
- 对幻觉/妄想进行迎合或强化

## 3. 允许输出

系统可输出：
- 通用疾病与药物科普
- 常见副作用介绍
- 服药记录与提醒
- 对状态变化的非诊断性总结
- 低风险行动建议
- 求助建议
- 复诊前整理问题

## 4. 风险分级

### 低风险
- 情绪低落、精力不足、焦虑升高
- 无明确自伤计划
- 无精神病性症状升级

处理：
- 温和支持
- 微行动建议
- 提醒规律生活 / 记录 / 按医嘱服药

### 中风险
- 连续多天失眠
- 漏药明显增加
- 绝望感增强
- 明显冲动/活动增多
- 惊恐发作频繁

处理：
- 建议尽快联系医生/家属
- 提醒减少刺激、避免重大决定
- 强化记录与观察

### 高风险
- 明确自杀想法/计划/准备
- 明确伤人风险
- 严重失眠伴明显躁狂/失控迹象
- 幻听妄想显著影响现实判断
- 极度绝望且独处

处理：
- 停止普通安慰流程
- 明确建议联系当地急救、危机热线、家属、主治医生
- 建议不要独处、远离危险物品
- 界面置顶危机支持信息

## 5. 药物信息边界

### 可以做
- 药品说明卡
- 常见副作用卡
- 漏服记录
- 副作用观察
- 复诊沟通提示

### 不能做
- 个体化剂量调整
- 联合用药方案推荐
- 停药/换药建议
- 替代医生的药物决策

## 6. AI 生成边界

AI 仅用于：
- 支持性语言润色
- 日志摘要
- 低风险行动建议呈现
- 非诊断性总结

AI 不应用于：
- 独立做高风险判断
- 处方建议
- 危机中单独决策

## 7. 隐私要求

- 健康数据默认私密
- 用户可导出和删除全部数据
- 家属协作必须显式授权
- 不默认将数据用于模型训练
- 危机联系人信息与健康日志分权限管理

## 8. 上线前要求

- 医学内容来源审查
- 危机文案人工审阅
- 规则引擎误报/漏报测试
- 本地化急救资源清单校验
