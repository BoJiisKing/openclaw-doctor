# Design Direction

## 设计主题

**Quiet Harbor / 安静的避风港**

这不是医院后台，也不是炫技科技产品。它应该像一个在困难时期仍然有秩序感、能让人稍微慢下来一点的空间。

## 关键词

- 安静
- 稳定
- 温和但不幼稚
- 有结构感
- 少刺激、少噪声
- 可信赖

## 视觉方向

- 偏柔和的深浅蓝灰 + 温暖点缀色
- 大圆角卡片，但避免过度可爱
- 明确的层级、留白和阅读节奏
- 少量、克制的动效
- 状态色：绿/橙/红，但整体不刺眼

## 体验原则

- 首先降低压迫感
- 让关键信息一眼能看到
- 让高风险内容始终比“阅读资料”优先
- 保持陪伴感，但不过度拟人化
